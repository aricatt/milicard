# 模块：供应商列表

- **参考截图**: `4_供应商.png`
- **URL**: `/live-base/base-data/suppliers`

---

## 1. 功能概述

此页面用于管理所有与本系统合作的供应商。用户可以查看供应商列表，并进行添加、编辑和删除操作。

---

## 2. 页面布局与功能

### 2.1 主要操作

- **添加供应商**:
  - 页面左上方应有一个“添加” (Add) 按钮。
  - 点击后，跳转到“添加供应商”页面。

### 2.2 供应商列表

- **数据展示**: 页面主体为一个表格，用于展示所有供应商信息。
- **分页**: 表格下方应有分页组件。
- **表格列**:
  - **ID**: 唯一标识符。
  - **编号**: 业务编号 (例如: `SUPPLIER-ST397NBHU6D`)。
  - **名称**: 供应商的官方名称。
  - **创建时间**: 记录的创建日期和时间。
  - **操作**:
    - **编辑 (Edit)**: 点击后，跳转到对应的编辑页面。
    - **删除 (Del)**: 点击后，应弹出确认对话框，确认后删除该记录。
