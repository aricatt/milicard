# 模块：基地列表

- **参考截图**: `1_基地.png`
- **URL**: `/live-base/base-data/bases`

---

## 1. 功能概述

此页面用于集中管理系统中的所有“基地”。用户可以在此页面查看所有基地的列表，并进行添加、编辑和删除操作。

---

## 2. 页面布局与功能

### 2.1 主要操作

- **添加基地**:
  - 页面左上方应有一个“添加” (Add) 按钮。
  - 点击后，跳转到“添加基地”页面。

### 2.2 基地列表

- **数据展示**: 页面主体为一个表格，用于展示所有基地信息。
- **分页**: 如果基地数量超过一页（例如，每页10条），表格下方应有分页组件。
- **表格列**:
  - **ID**: 数据库中的唯一标识符。
  - **编号**: 系统的业务编号 (例如: `BASE-83Q6731DP7J`)。
  - **名称**: 基地的名称 (例如: `火山`, `杭州基地`)。
  - **创建时间**: 该记录的创建日期和时间。
  - **操作**:
    - **编辑 (Edit)**: 点击后，跳转到该基地的“编辑”页面，页面应预先填充该基地的现有信息。
    - **删除 (Del)**: 点击后，应弹出确认对话框，防止误操作。确认后，删除该基地记录。

---
