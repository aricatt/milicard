## ❌ 刚修正的表格

### 7. 采购订单导入表

**转换脚本原字段**：
```
采购单号, 供应商, 采购品类, 厂商, 产品盒数, 产品规格, 产品重量(kg), 单价(RMB), 采购总金额, 下单时间, 发货时间, 货运单号, 状态
```

**系统需要的字段**：
```
采购日期, 采购编号, 品类, 商品名称, 供应商, 采购箱, 采购盒, 采购包, 拿货单价箱, 实付金额
```

**已修正为**：
```
采购日期, 采购编号, 品类, 商品名称, 供应商, 采购箱, 采购盒, 采购包, 拿货单价箱, 实付金额
```

**注意事项**：
- 拿货单价箱和实付金额需要手动填写，格式：`[VND]金额` 或 `[CNY]金额`
- 原始数据缺少箱数和包数，默认填0
- 原始数据的单价和总金额无法直接使用（没有货币标记）
